css for emacs manuals

I've been working on a css for the emacs related manuals. It is just a tweak away in the makefile to get the build process to insert the adequate meta tag, you can even add the line manually for a single page manual, and then write the external css file to have something visually more pleasing. I just committed the one I had been working for, after a few modifications tonight. It is here: emacs.css. I have decided not to modify the texinfo output and just work directly on the original HTML file. It is not the best user experience and my choice of parameters can be questioned, but it is available now and it certainly makes the elisp reference easier for me to read.
