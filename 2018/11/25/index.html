What kept me busy all Sunday

  • I wrote an emacs lisp script to automate the generation of daily templates.

I'll document that another day. But suffice it to say that it took my whole Sunday, from 9am to 2am. With breaks in between to eat, play with the kids, watch a bit of Kung-fu Hussle, have coffee with friends, etc. But it was still a full day at the keyboard and I'm exhausted.

Also, it works. I'll post the code another day.

The code is here: adventuresintechland.el It is an emacs file where the function "dailyIndex" creates the index.html file for the day, with all the relevant links. It mostly is a list of variable assignments and one big format command that puts all the variable values in place in the html. Then it opens the file for editing.
